# Tauri + Leptos

This template should help get you started developing with Tauri and Leptos.

[create project](https://v2.tauri.app/ja/start/create-project/)

```shell
cargo install create-tauri-app --locked
cargo create-tauri-app
```

run

```shell
cargo tauri dev
```
